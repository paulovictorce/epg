# EPG
The EGP Solution - A Rest API

# Technologies
- Java 8  
- Spring Boot (2.7.5)  
- Maven (3.8.6)  
- Spring Boot Starter Data JPA  
- H2 Database  
- Lombok  
- Swagger  
- Junit and Mockito (for unity tests)  

# Documentation
After running the project locally, go to http://localhost:8080/swagger-ui.html to view all available endpoints.
